# References

This folder stores external or generated reference artifacts used to guide conversion behavior.

- `fritzing-parts/`: Existing `.fzp` files used as structure references.
- `kicad-exports/`: SVG exports from KiCad used for geometry and footprint checks.
- `kicad-projects/`: Small KiCad project fixtures used for parser and mapping development.
- `samples/`: Miscellaneous sketches, experiments, and sample assets.

Keep source code out of this directory; it should only contain artifacts and notes.
